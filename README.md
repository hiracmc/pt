# つくりかた
## Renderの場合
1.このリポジトリをフォークする

2.renderにサインイン

3.+newから「Static Site」を選択
![画像](https://hiracmc.github.io/assets/pt1.jpeg)

4.Public Git Repositoryを選択して、先ほどフォークしたリポジトリのurlを入力

5.Publish Directoryに「.」を打ち込む

6.保存して完成！🎉
